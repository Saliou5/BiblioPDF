Comment télécharger un PDF

Tutoriel vidéo

Étapes pour télécharger un PDF :

  1. Recherchez le PDF que vous souhaitez télécharger
  2. Cliquez sur le bouton "Voir" pour visualiser le PDF
  3. Dans la visionneuse PDF, cliquez sur l'icône de téléchargement
  4. Choisissez l'emplacement où enregistrer le fichier
  5. Attendez que le téléchargement se termine

Note : Vous devez être connecté pour pouvoir télécharger les PDF. Si vous n'avez pas de compte, vous pouvez en créer un gratuitement.
